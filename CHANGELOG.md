# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-01-22

### Added
- **Map Beacon Selection**: Users can now open a map modal to select and transmit a custom location beacon.
- **Leaflet Integration**: Integrated Leaflet.js for map rendering.
- **Improved Terminal**: Raw beacon terminal is now visible by default for better transparency.

### Changed
- Updated `README.md` to a modern, comprehensive format.
- Refined UI styles for modal interactions.

## [1.0.0] - 2026-01-20

### Added
- Initial Release of APRS Web Messenger.
- Real-time messaging via APRS-IS.
- Glassmorphism user interface.
- Basic GPS location beaconing.
- Contact management system.
- Settings for callsign, passcode, and symbol customization.
