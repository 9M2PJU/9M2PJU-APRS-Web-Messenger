# Changelog

## [v1.5.0] - 2026-01-20
### Added
- **Privacy Lock**: New local Access Password system to secure the application.
- **Settings Menu**: Configuration portal for APRS-IS passcode, coordinates, and beacons.
- **Raw Packet Console**: Live terminal-style monitor for technical data streams.
- **AIS-ST Beaconing**: Automatic status beaconing with configurable intervals.
- **Radar Mini-Map**: Dynamic visual "ping" animation for the right panel.
- **Quick Macros**: Pre-set buttons for common APRS shorthand (73, QRV, QSL).
- **Management Tools**: Ability to delete specific contacts and clear entire chat histories.
- **Notification Sounds**: Retro-futuristic beep on incoming messages.

### Fixed
- Improved Blob-to-text decoding for binary WebSocket responses.
- Explicit server-side login verification handling.

### Changed
- Refined **Arch Linux** theme with JetBrains Mono and command-center layout.
- Separated local authentication from APRS-IS network credentials.
