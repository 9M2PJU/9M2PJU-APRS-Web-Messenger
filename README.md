# 9M2PJU APRS Web Messenger

A futuristic, high-performance web application for sending and receiving APRS messages via APRS-IS. Designed with an **Arch Linux** aesthetic, featuring deep-blue glassmorphism and JetBrains Mono typography.

- **Quick Macros**: One-tap communication (73, QRV, QSL).

- **Direct APRS-IS Integration**: Connects via `wss://ametx.com:8888`.
- **Intelligent Filtering**: Server-side filtering to keep your stream quiet.
- **Auto-Beaconing**: Keeps your station visible on the map automatically.
- **Full History Management**: Delete contacts or clear conversations with one click.
- **Persistent Storage**: All settings and messages are saved to `localStorage`.
